# Contributing to ABC Server

Thank you for your interest in contributing to the ABC Server security client!

## Code of Conduct

Please be respectful and professional in all interactions.

## How to Contribute

1. **Report Bugs**: Open an issue describing the bug, steps to reproduce, and the expected vs. actual behavior.
2. **Suggest Features**: Submit a feature request describing the proposed feature and why it would be beneficial.
3. **Submit Pull Requests**:
   - Ensure all tests pass (`./gradlew test`).
   - Follow the established package conventions (`com.inscopelabs.abxmcp.*`).
   - Secure critical code blocks and avoid hardcoded secrets.
